# Проект Окна

Многостраничный проект с автоматической компиляцией SCSS в CSS и минификацией.

## Установка

```bash
npm install
```

## Запуск разработки

```bash
npm run dev
```

Эта команда запустит:
- Автоматическую компиляцию SCSS в CSS при изменении файлов
- Локальный сервер на http://localhost:3000

## Работа только с SCSS

```bash
npm run scss
```

## Сборка проекта

```bash
npm run build
```

Создаст скомпилированные `.css` и минифицированные `.min.css` файлы в папке `dist/css/`

## Структура проекта

```
├── src/
│   ├── scss/           # SCSS исходники
│   │   ├── abstracts/  # Переменные, миксины, функции
│   │   ├── base/       # Базовые стили, сброс, типография
│   │   ├── components/ # Компоненты (кнопки, формы и т.д.)
│   │   ├── layout/     # Макет (header, footer, grid)
│   │   ├── pages/      # Стили для конкретных страниц
│   │   └── main.scss   # Главный файл импорта
│   └── js/             # JavaScript файлы
├── dist/
│   └── css/            # Скомпилированные CSS файлы
├── assets/
│   ├── images/         # Изображения
│   ├── fonts/          # Шрифты
│   └── icons/          # Иконки
└── pages/              # HTML страницы (кроме index.html)
```
